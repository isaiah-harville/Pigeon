# PigeonRelay Source Map

  challenge-response mailbox authentication, publish/subscribe/ack protocol, and
  health endpoint.
- `Cargo.toml` — Rust package metadata and dependencies.
- `Dockerfile` — distroless multi-arch image build.
- `README.md` — deployment, protocol, and relay threat model.
